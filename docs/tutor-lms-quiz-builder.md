# Quiz Builder

Quiz Builder adalah fitur Tutor LMS untuk membuat kuis interaktif yang menguji pemahaman siswa. Tersedia **10 tipe pertanyaan modern** untuk berbagai kebutuhan evaluasi pembelajaran.

---

## Memulai Quiz Builder

1. Buka **Course Builder** kursus Anda
2. Pilih topik yang ingin ditambahkan kuis
3. Klik tombol **"Kuis"** di bawah topik
4. Masukkan **nama** dan **deskripsi** kuis
5. Mulai tambahkan pertanyaan

![Tampilan awal Quiz Builder](https://is3.cloudhost.id/elsweb/2022/07/image-8.png)

---

## 10 Tipe Pertanyaan

![Pilihan tipe pertanyaan](https://is3.cloudhost.id/elsweb/2022/07/image-9.png)

| Tipe | Deskripsi | Cocok untuk |
|------|-----------|-------------|
| **Benar / Salah** | Siswa memilih benar atau salah | Konsep fundamental |
| **Pilihan Tunggal** | Satu jawaban benar dari beberapa opsi | Fakta dan definisi |
| **Pilihan Ganda** | Satu atau lebih jawaban benar | Konsep kompleks |
| **Open-Ended / Esai** | Jawaban deskriptif panjang dengan batas karakter | Analisis mendalam |
| **Fill in the Blanks** | Melengkapi kalimat yang tidak lengkap | Hafalan & pemahaman |
| **Jawaban Singkat** | Kalimat jawaban singkat dan deskriptif | Ringkasan konsep |
| **Mencocokkan** | Pasangkan dua konsep atau kalimat | Relasi & asosiasi |
| **Pencocokan Gambar** | Pasangkan dua gambar yang berhubungan | Visual & desain |
| **Image Answering** | Identifikasi jawaban berdasarkan gambar | Visual recognition |
| **Pengurutan** | Susun pilihan sesuai urutan yang benar | Proses & tahapan |

## Pengaturan Per Pertanyaan

![Pengaturan pertanyaan individual](https://is3.cloudhost.id/elsweb/2022/07/image-10.png)

| Pengaturan | Fungsi |
|-----------|--------|
| **Diperlukan Jawaban** | Tandai pertanyaan sebagai wajib dijawab |
| **Acak** | Acak urutan pilihan jawaban |
| **Poin** | Tentukan nilai poin kustom untuk pertanyaan ini |
| **Tampilkan Poin** | Aktifkan tampilan poin di halaman kuis |

## Pengaturan Umum Kuis

![Pengaturan umum kuis](https://is3.cloudhost.id/elsweb/2022/07/image-11.png)

### Waktu & Tampilan

| Pengaturan | Fungsi |
|-----------|--------|
| **Batas Waktu** | Durasi maksimal pengerjaan kuis |
| **Sembunyikan Tampilan Waktu** | Hilangkan penghitung waktu dari layar siswa |
| **Quiz Auto Start** | Kuis langsung mulai saat halaman dimuat |
| **Sembunyikan Nomor Pertanyaan** | Hilangkan penomoran pertanyaan |

### Evaluasi & Nilai

| Pengaturan | Fungsi |
|-----------|--------|
| **Mode Umpan Balik** | Kapan feedback ditampilkan (saat menjawab / setelah selesai) |
| **Upaya Diizinkan** | Berapa kali siswa dapat mengulang kuis |
| **Passing Grade** | Nilai minimum untuk dinyatakan lulus (%) |
| **Pertanyaan Maksimal** | Jumlah soal yang disajikan dari total bank soal |

### Format & Layout

| Pengaturan | Fungsi |
|-----------|--------|
| **Tata Letak Pertanyaan** | Format tampilan halaman kuis |
| **Urutan Pertanyaan** | Urutan penyajian soal (tetap / acak) |
| **Batas Karakter Jawaban Singkat** | Limit karakter untuk tipe short answer |
| **Batas Karakter Esai** | Limit karakter untuk tipe open-ended |

### Import & Export

| Pengaturan | Fungsi |
|-----------|--------|
| **Impor Kuis** | Import pertanyaan dari file CSV |

## Tips Membuat Kuis yang Efektif

{% hint style="success" %}
**Rekomendasi:**
- Gunakan **1 kuis per topik** untuk evaluasi per bagian
- Campurkan tipe pertanyaan untuk variasi
- Set **passing grade 70%** sebagai standar minimum
- Aktifkan **"Acak"** untuk mencegah siswa saling menyontek
- Gunakan **batas waktu** untuk mendorong pemahaman yang sesungguhnya
{% endhint %}
