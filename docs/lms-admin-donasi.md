# Donasi

Modul Donasi memungkinkan lembaga Anda menggalang dana lewat kampanye — cocok untuk lembaga sosial, pendidikan, atau dakwah. Grup **Donasi** berisi empat menu: **Kampanye Donasi**, **Kategori Donasi**, **Transaksi Donasi**, dan **Laporan Dana**.

{% hint style="warning" %}
Donasi adalah **fitur tambahan (add-on)** yang perlu **diaktifkan** terlebih dahulu untuk lembaga Anda. Jika grup **Donasi** belum muncul di sidebar, ajukan aktivasi lewat menu **Pengajuan Donasi** atau hubungi tim kami. Sampai aktif, menu-menu di halaman ini tidak akan terlihat.
{% endhint %}

---

## Prasyarat

- Modul Donasi sudah aktif untuk lembaga Anda
- Anda bisa login sebagai **Super Admin** atau **Admin** — lihat [Ikhtisar dan Akses Panel Admin](lms-admin-pengenalan.md)

---

## 1. Membuat Kategori Donasi

Buat kategori lebih dulu agar kampanye bisa dikelompokkan. Klik **Donasi → Kategori Donasi → Create**, isi nama kategori, dan pastikan berstatus aktif.

Contoh kategori: "Pendidikan", "Wakaf", "Bantuan Bencana".

---

## 2. Membuat Kampanye Donasi

Klik **Donasi → Kampanye Donasi → Create**.

![Membuat kampanye donasi](https://GANTI-DENGAN-SCREENSHOT)

| Field | Keterangan |
|-------|-----------|
| **Kategori** | Kategori donasi (dari langkah 1) |
| **Pengelola** | Penanggung jawab kampanye (Super Admin/Admin) |
| **Judul Kampanye** | Nama kampanye |
| **Slug** | Versi URL dari judul |
| **Deskripsi Singkat** & **Deskripsi Lengkap** | Cerita dan tujuan kampanye |
| **Thumbnail** | Gambar utama kampanye |
| **Target Dana (Rp)** | Nominal target penggalangan |
| **Status** | Draft / Aktif / Selesai |
| **Mulai** & **Berakhir** | Periode kampanye |
| **Meta Title** & **Meta Description** | Untuk SEO |

{% hint style="success" %}
Kampanye dengan cerita yang jelas, gambar yang kuat, dan target realistis cenderung lebih menarik donatur. Tulis deskripsi lengkap yang menjelaskan ke mana dana akan disalurkan.
{% endhint %}

Simpan sebagai **Draft** dulu, lalu ubah ke **Aktif** saat siap dipublikasikan.

---

## 3. Memantau Transaksi Donasi

Klik **Donasi → Transaksi Donasi** untuk melihat donasi yang masuk. Seperti pada order kursus, donasi lewat **transfer manual** perlu dikonfirmasi setelah donatur mengunggah bukti transfer — cocokkan dengan mutasi rekening sebelum mengonfirmasi.

{% hint style="warning" %}
Konfirmasikan donasi hanya setelah dana benar-benar masuk ke rekening. Angka yang terkonfirmasi akan terhitung ke dalam capaian kampanye dan laporan.
{% endhint %}

---

## 4. Laporan Dana

Klik **Donasi → Laporan Dana** untuk melihat rekap dana per kampanye — total terkumpul, capaian terhadap target, dan penyalurannya. Gunakan laporan ini untuk transparansi kepada donatur.

---

## Checklist Donasi

- [ ] Modul Donasi sudah aktif untuk lembaga Anda
- [ ] Kategori donasi sudah dibuat sebelum kampanye
- [ ] Kampanye punya cerita, gambar, target, dan periode yang jelas
- [ ] Transaksi transfer manual dikonfirmasi setelah dana masuk
- [ ] Laporan dana dipantau untuk transparansi
