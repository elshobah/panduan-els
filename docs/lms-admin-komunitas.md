# Komunitas dan Diskusi

Fitur **Diskusi** (grup **Komunitas**) adalah forum tempat peserta bertanya, berbagi, dan berdiskusi. Sebagai admin, tugas Anda adalah **memoderasi** — menjaga agar diskusi tetap tertib, relevan, dan sopan.

---

## Prasyarat

- Anda bisa login sebagai **Super Admin** atau **Admin** — lihat [Ikhtisar dan Akses Panel Admin](lms-admin-pengenalan.md)

---

## 1. Meninjau Diskusi

Klik **Komunitas → Diskusi**. Tabel menampilkan semua thread beserta penulis, channel, kategori, jumlah balasan, dan penanda status (pin/kunci).

![Daftar diskusi](https://GANTI-DENGAN-SCREENSHOT)

Gunakan filter **Kategori**, **Dipinned**, dan **Dikunci** untuk menyaring. Klik **Detail** untuk membaca isi thread lengkap dengan balasannya.

---

## 2. Aksi Moderasi

Saat membuka atau mengedit sebuah thread, tersedia dua sakelar dan opsi hapus:

| Aksi | Fungsi | Kapan dipakai |
|------|--------|---------------|
| **Pin** | Menyematkan thread di atas | Untuk pengumuman atau diskusi penting yang ingin selalu terlihat |
| **Kunci Thread** | Menghentikan balasan baru | Untuk diskusi yang sudah selesai atau mulai memanas |
| **Hapus** | Menghapus thread | Untuk spam, konten melanggar, atau duplikat |

{% hint style="info" %}
**Kunci** lebih ramah daripada **Hapus** untuk diskusi yang sudah tuntas — isinya tetap bisa dibaca sebagai arsip, hanya tidak bisa ditambah balasan.
{% endhint %}

{% hint style="danger" %}
**Menghapus** thread bersifat permanen dan ikut menghapus balasannya. Gunakan hanya untuk konten yang benar-benar melanggar. Jika ragu, cukup **Kunci** thread-nya.
{% endhint %}

---

## 3. Tips Moderasi yang Sehat

- **Sematkan** aturan komunitas atau FAQ sebagai thread pin permanen.
- Tanggapi pertanyaan yang belum terjawab agar peserta merasa didengar.
- Konsisten menerapkan aturan — moderasi yang adil membangun kepercayaan.
- Untuk pelanggaran berulang oleh pengguna tertentu, pertimbangkan tindakan di [Mengelola Pengguna dan Peran](lms-admin-pengguna-peran.md).

{% hint style="success" %}
Aktifkan notifikasi **Diskusi/Pertanyaan Baru** di [Pengaturan Umum](lms-admin-pengaturan.md) agar Anda segera tahu saat ada thread baru yang perlu ditanggapi.
{% endhint %}

---

## Checklist Moderasi

- [ ] Aturan komunitas disematkan (pin) sebagai thread permanen
- [ ] Diskusi rutin ditinjau untuk spam atau konten melanggar
- [ ] Menggunakan **Kunci** untuk diskusi selesai, **Hapus** hanya untuk pelanggaran
- [ ] Notifikasi diskusi baru diaktifkan
