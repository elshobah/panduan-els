## Prasyarat

Sebelum menambahkan produk, pastikan:

- Plugin **WooCommerce** sudah terinstal dan aktif di WordPress Anda
- Anda sudah login ke dashboard WordPress (lihat [Cara Login Admin](cara-login-wordpress.md))

## Langkah-Langkah

### 1. Buka Menu Produk

Di sidebar WordPress, pilih **Products** → **Add New**.

### 2. Isi Detail Produk

| Field | Keterangan |
|-------|-----------|
| **Product name** | Nama produk yang akan ditampilkan |
| **Description** | Deskripsi lengkap produk |
| **Short description** | Ringkasan singkat (tampil di halaman produk) |
| **Product image** | Gambar utama produk |
| **Product gallery** | Gambar tambahan produk |

### 3. Atur Harga

Di panel **Product data**:

- **Regular price** — harga normal
- **Sale price** — harga diskon (opsional)

### 4. Atur Kategori & Tag

- Pilih **kategori produk** yang sesuai di panel kanan
- Tambahkan **tags** untuk memudahkan pencarian

### 5. Publish

Klik **Publish** untuk menampilkan produk di toko online.
