# Menu Announcement

Tutor LMS menyediakan bagian tersendiri untuk mengelola semua pengumuman kursus. Fitur pengumuman **diaktifkan secara otomatis** dan dapat dinonaktifkan melalui pengaturan jika diperlukan.

**Akses via:** WordPress Admin → **Tutor LMS** → **Announcement**

{% hint style="info" %}
Untuk menonaktifkan fitur pengumuman: Tutor LMS → Pengaturan → Kursus → nonaktifkan opsi Announcement.
{% endhint %}

## Fitur Filter & Pencarian

### Pencarian
Gunakan kotak pencarian untuk menemukan pengumuman spesifik menggunakan kata kunci. Hasil akan ditampilkan secara otomatis.

### Filter Kursus
Dropdown yang menampilkan seluruh kursus aktif — filter pengumuman berdasarkan kursus tertentu.

### Sort By (Urutkan)
Urutkan daftar pengumuman secara **ascending** (A–Z / terlama) atau **descending** (Z–A / terbaru).

### Filter Tanggal
Tampilkan pengumuman berdasarkan periode tanggal tertentu.

## Membuat Pengumuman Baru

1. Klik tombol **"Tambah Baru"** di pojok kanan atas
2. Isi form pengumuman:

| Field | Keterangan |
|-------|-----------|
| **Pilih Kursus** | Tentukan kursus yang akan menerima pengumuman |
| **Judul Pengumuman** | Judul singkat dan deskriptif |
| **Ringkasan** | Isi pengumuman secara lengkap |
| **Notifikasi Siswa** | Centang untuk mengirim notifikasi ke semua siswa terdaftar |

3. Klik **Simpan / Publish**

{% hint style="success" %}
Aktifkan opsi **"Notifikasi Siswa"** agar semua peserta kursus mendapat pemberitahuan via email secara otomatis.
{% endhint %}
