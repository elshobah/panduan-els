# Cara Membuat Postingan di WordPress

Panduan ini menjelaskan cara membuat artikel atau postingan baru di WordPress secara lengkap — mulai dari menulis konten, menambahkan gambar, mengatur kategori, hingga mempublikasikan.

{% embed url="https://www.youtube.com/watch?v=JiriYSqLJkU" %}

---

## Langkah-Langkah

### 1. Akses Dashboard WordPress

Login ke dashboard WordPress Anda terlebih dahulu. Lihat panduan [Cara Login Admin di WordPress](cara-login-wordpress.md) jika belum tahu caranya.

### 2. Buka Halaman Posting Baru

Di menu sidebar kiri, pilih **Posts** → klik **Add New**.

![Navigasi ke Posts > Add New](https://is3.cloudhost.id/elsweb/2022/07/image-13.png)

### 3. Isi Judul dan Konten

Anda akan diarahkan ke halaman **Add New Post**.

- Masukkan **judul artikel** pada kolom **Add title**
- Mulai menulis konten pada area editor di bawahnya

![Halaman editor postingan WordPress](https://is3.cloudhost.id/elsweb/2022/07/image-14.png)

### 4. Format Teks

Gunakan toolbar formatting untuk mempercantik konten:

![Toolbar formatting WordPress](https://is3.cloudhost.id/elsweb/2022/07/image-15.png)

| Opsi | Fungsi |
|------|--------|
| **Bold / Italic** | Tebalkan atau miringkan teks |
| **Bullets / Numbering** | Daftar tidak berurutan / berurutan |
| **Alignment** | Rata kiri, tengah, kanan, atau justify |
| **Blockquote** | Kutipan dengan tampilan khusus |
| **Strikethrough** | Teks dengan coretan |
| **Text Color** | Warna teks kustom |

### 5. Tambahkan Media

Klik tombol **Add Media** untuk menyisipkan gambar, video, atau audio ke dalam konten.

![Tombol Add Media](https://is3.cloudhost.id/elsweb/2022/07/image-16.png)

### 6. Konfigurasi Media

Di halaman **Add Media**, klik **Select Files** untuk mengunggah file dari komputer Anda.

![Halaman Add Media](https://is3.cloudhost.id/elsweb/2022/07/image-17.png)

Setelah file terunggah, isi informasi berikut:

| Field | Keterangan |
|-------|-----------|
| **Alt Text** | Deskripsi gambar untuk SEO dan aksesibilitas — **wajib diisi** |
| **Title** | Judul gambar |
| **Caption** | Teks keterangan di bawah gambar |
| **Description** | Deskripsi panjang gambar |

Atur alignment dan ukuran gambar sesuai kebutuhan, lalu klik **Insert into post**.

{% hint style="success" %}
**Tips SEO:** Selalu isi **Alt Text** dengan deskripsi yang relevan. Ini membantu Google memahami isi gambar Anda.
{% endhint %}

### 7. Atur Kategori

Di sidebar kanan, temukan panel **Categories**.

![Panel Categories](https://is3.cloudhost.id/elsweb/2022/07/image-18.png)

- Pilih kategori yang sudah ada dengan mencentangnya
- Atau buat kategori baru dengan klik **+ Add New Category**

### 8. Tambahkan Tags

Di panel **Tags**, masukkan tag yang relevan dengan konten dan klik **Add**.

Tags membantu pengunjung menemukan konten terkait dan memperkuat struktur SEO internal.

### 9. Edit Permalink (Slug)

URL artikel default biasanya tidak optimal untuk SEO. Ubah dengan:

1. Klik tombol **Edit** di bawah kolom judul
2. Tulis ulang slug menjadi singkat, deskriptif, dan mengandung keyword utama
3. Klik **OK**

**Contoh:**
- ❌ `domain.com/?p=123`
- ✅ `domain.com/tips-seo-wordpress`

### 10. Set Featured Image

Featured image adalah gambar utama yang mewakili postingan.

1. Klik **Set featured image** di panel sidebar kanan
2. Unggah atau pilih gambar dari media library
3. Klik **Set featured image**

![Set Featured Image](https://is3.cloudhost.id/elsweb/2022/07/image-19.png)

### 11. Publikasikan

Di panel **Publish** (pojok kanan atas), Anda memiliki beberapa opsi:

![Panel Publish](https://is3.cloudhost.id/elsweb/2022/07/image-20.png)

| Opsi | Fungsi |
|------|--------|
| **Save Draft** | Simpan sebagai draf, belum dipublikasikan |
| **Preview** | Lihat tampilan artikel sebelum dipublikasikan |
| **Status** | Draft / Pending Review / Published |
| **Visibility** | Public / Password Protected / Private |
| **Schedule** | Jadwalkan publikasi di tanggal & waktu tertentu |
| **Publish** | Publikasikan sekarang |

Klik **Publish** untuk menerbitkan artikel secara langsung.

### 12. Selesai

Postingan Anda telah berhasil diterbitkan dan dapat diakses oleh pengunjung website.

---

## Checklist Sebelum Publish

{% hint style="info" %}
Pastikan semua poin ini sudah diisi sebelum menekan tombol **Publish**:
{% endhint %}

- [ ] Judul artikel menarik dan mengandung keyword utama
- [ ] Konten lengkap dan tidak ada typo
- [ ] Alt Text semua gambar sudah diisi
- [ ] Kategori sudah dipilih
- [ ] Tags relevan sudah ditambahkan
- [ ] Permalink/slug sudah dioptimasi
- [ ] Featured image sudah diset
- [ ] Preview sudah dilihat dan tampilan sesuai
