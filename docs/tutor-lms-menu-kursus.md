# Menu Kursus

Menu Kursus adalah pusat pengelolaan semua kursus yang ada di platform LMS Anda. Dari sini Anda dapat melihat, mengedit, menduplikat, dan menghapus kursus.

**Akses via:** WordPress Admin → **Tutor LMS** → **Kursus**

---

## Tampilan Daftar Kursus

![Tampilan daftar kursus di Tutor LMS](https://elshobah.com/wp-content/uploads/2022/07/image.png)

Setiap kursus ditampilkan dalam tabel dengan kolom:
- Judul kursus
- Kategori
- Penulis/instruktur
- Tag
- Harga
- Status publikasi

---

## Opsi Pengelolaan Kursus

Arahkan kursor ke nama kursus untuk memunculkan opsi aksi:

![Opsi pengelolaan kursus](https://is3.cloudhost.id/elsweb/2022/07/image-1.png)

### Edit
Buka panel editor backend lengkap untuk melakukan perubahan komprehensif pada kursus — konten, pengaturan, harga, modul, dll.

### Quick Edit
Lakukan perubahan cepat tanpa harus membuka halaman editor penuh.

Opsi yang tersedia via Quick Edit:

| Field | Keterangan |
|-------|-----------|
| **Judul** | Ubah nama kursus |
| **Slug** | Edit URL kursus |
| **Tanggal** | Ubah tanggal publikasi |
| **Kata Sandi** | Proteksi kursus dengan password |
| **Kategori** | Pindah atau tambah kategori |
| **Tags** | Edit tag kursus |
| **Status** | Published / Draft / Pending Review |

### Trash (Sampah)
Memindahkan kursus ke tempat sampah dan **otomatis mencabut publikasi**. Kursus dapat dipulihkan dari menu Trash selama belum dihapus permanen.

### View (Lihat)
Buka halaman kursus di frontend website untuk melihat tampilan yang dilihat calon siswa.

### Duplicate (Duplikat)
Menyalin seluruh informasi kursus — judul, deskripsi, modul, lesson, pengaturan — ke editor baru sebagai **draft**. Berguna untuk membuat kursus serupa tanpa mulai dari nol.

{% hint style="success" %}
Gunakan fitur **Duplicate** saat membuat seri kursus (Bagian 1, Bagian 2, dst.) dengan struktur yang mirip.
{% endhint %}
