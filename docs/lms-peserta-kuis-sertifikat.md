# Kuis, Nilai, dan Sertifikat

Sebagian kursus memiliki kuis untuk menguji pemahaman, dan memberi **sertifikat** saat kursus selesai. Panduan ini menjelaskan cara mengerjakan kuis, melihat nilai, dan mengunduh sertifikat Anda.

---

## Prasyarat

- Anda sedang mengikuti kursus — lihat [Belajar di Kursus](lms-peserta-belajar.md)

---

## Bagian 1 — Mengerjakan Kuis

### 1. Mulai Kuis

Saat sebuah materi berupa kuis, akan tampil ajakan **"Kerjakan kuis ini untuk menyelesaikan materi."** Klik **Mulai Kuis**.

![Halaman mulai kuis](https://GANTI-DENGAN-SCREENSHOT)

### 2. Menjawab Soal

Soal ditampilkan satu per satu dengan penanda **"Soal {n} dari {total}"** dan berapa soal yang sudah dijawab. Gunakan tombol **Sebelumnya** dan **Berikutnya** untuk berpindah. Setelah semua terjawab, klik **Kumpulkan Jawaban**.

### 3. Melihat Hasil

Setelah dikumpulkan, hasil langsung tampil:

| Info | Arti |
|------|------|
| **LULUS** / **TIDAK LULUS** | Status kelulusan Anda |
| **Skor (%)** | Nilai yang Anda peroleh |
| **Nilai minimum lulus: {x}%** | Ambang batas kelulusan |

Klik **Lihat Hasil** untuk rincian. Jika belum lulus, klik **Ulangi Kuis** untuk mencoba lagi.

{% hint style="info" %}
Kuis boleh diulang. Ambil waktu untuk mengulang materi terkait sebelum mencoba lagi agar hasilnya lebih baik.
{% endhint %}

---

## Bagian 2 — Melihat Nilai (Gradebook)

Buka **Nilai & Sertifikat → Gradebook** di sidebar untuk melihat rekap nilai kuis Anda di seluruh kursus. Ini membantu Anda memantau perkembangan belajar.

![Halaman Gradebook](https://GANTI-DENGAN-SCREENSHOT)

---

## Bagian 3 — Sertifikat

Saat kursus mencapai **100%** (semua materi selesai, dan lulus kuis bila disyaratkan), sertifikat penyelesaian otomatis dibuat untuk Anda.

### 1. Membuka Sertifikat

Buka **Nilai & Sertifikat → Sertifikat** di sidebar. Daftar sertifikat Anda akan muncul. Klik salah satu untuk membuka halaman **Sertifikat Penyelesaian** yang menampilkan:

- **Diberikan kepada {nama Anda}**
- **telah berhasil menyelesaikan {judul kursus}**
- Tanggal terbit dan **No. {nomor sertifikat}**

![Halaman sertifikat](https://GANTI-DENGAN-SCREENSHOT)

### 2. Mengunduh PDF

Klik **Unduh PDF** untuk menyimpan sertifikat. Anda juga bisa klik **Buka Kursus** untuk kembali ke materi.

{% hint style="info" %}
Jika tombol masih bertuliskan **PDF Belum Tersedia**, berkas sertifikat sedang disiapkan. Tunggu beberapa saat lalu muat ulang halaman.
{% endhint %}

{% hint style="success" %}
Pastikan **Nama Lengkap** di profil Anda sudah benar **sebelum** menyelesaikan kursus — nama itulah yang tercetak di sertifikat. Atur di [Akun Saya](lms-peserta-akun.md).
{% endhint %}

### 3. Verifikasi Sertifikat

Setiap sertifikat punya **nomor unik** yang tercetak pada PDF. Nomor ini bisa digunakan untuk memverifikasi keaslian sertifikat secara publik.

---

## Checklist

- [ ] Mengerjakan kuis dan menekan **Kumpulkan Jawaban**
- [ ] Memahami skor dan **Nilai minimum lulus**
- [ ] Mengulang kuis bila belum lulus
- [ ] Nama di profil sudah benar sebelum kursus selesai
- [ ] Mengunduh sertifikat lewat **Unduh PDF**
