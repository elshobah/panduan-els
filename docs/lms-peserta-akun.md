# Akun Saya

Halaman ini membantu Anda menjaga data akun tetap benar dan memantau aktivitas Anda — mulai dari profil, alamat, riwayat pesanan, hingga notifikasi.

---

## Prasyarat

- Anda sudah punya akun dan bisa masuk — lihat [Membuat Akun dan Masuk](lms-peserta-daftar-login.md)

---

## Profil Saya

Buka **Profil Saya** dari menu akun. Halaman ini terdiri dari beberapa kartu.

![Halaman Profil Saya](https://GANTI-DENGAN-SCREENSHOT)

### Informasi Dasar

Berisi **Nama** dan data dasar Anda.

{% hint style="warning" %}
Pastikan **Nama** Anda benar — nama ini yang tercetak pada **sertifikat** penyelesaian kursus. Perbaiki sebelum menyelesaikan kursus. Lihat [Kuis, Nilai, dan Sertifikat](lms-peserta-kuis-sertifikat.md).
{% endhint %}

### Nomor WhatsApp

Isi **Nomor WhatsApp** Anda. 

{% hint style="info" %}
Nomor ini **digunakan untuk undangan grup WhatsApp webinar & event**. Jika Anda mengikuti kursus webinar, pastikan nomor ini aktif agar tidak ketinggalan undangan grup.
{% endhint %}

### Alamat Pengiriman

Isi **Alamat Pengiriman** — provinsi, kabupaten/kota, kecamatan, hingga kelurahan (dipilih berjenjang), plus alamat lengkap dan kode pos.

{% hint style="info" %}
Alamat ini dipakai untuk **pengiriman merchandise dan hadiah kursus**. Isi bila kursus Anda menjanjikan barang fisik.
{% endhint %}

### Foto Profil (Avatar)

Unggah foto profil Anda agar tampil di dashboard dan diskusi.

### Ganti Password

Pada kartu **Ganti Password**, masukkan password baru untuk memperbarui kata sandi akun Anda.

{% hint style="success" %}
Gunakan password yang kuat dan tidak dipakai di layanan lain. Jangan bagikan password kepada siapa pun.
{% endhint %}

---

## Riwayat Pesanan

Buka **Pesanan Saya** di sidebar untuk melihat **Riwayat Pesanan** — semua pembelian kursus Anda beserta statusnya.

| Status pesanan | Status bayar | Aksi tersedia |
|----------------|--------------|---------------|
| **Selesai** / **Pending** / **Dibatalkan** | **Lunas** / **Belum Bayar** / **Gagal** / **Dikembalikan** | **Bayar Sekarang**, **Detail**, **Belajar** |

{% hint style="info" %}
Jika ada pesanan yang belum dibayar, klik **Bayar Sekarang** di sini untuk melanjutkan pembayaran. Lihat [Menyelesaikan Pembayaran](lms-peserta-pembayaran.md).
{% endhint %}

---

## Notifikasi

Buka **Notifikasi** untuk melihat pemberitahuan — konfirmasi pembayaran, pengumuman, dan aktivitas lain. Anda bisa menandai semua sebagai telah dibaca.

---

## Ulasan Saya

Buka **Ulasan Saya** untuk melihat dan menulis ulasan kursus yang Anda ikuti. Ulasan membantu peserta lain memilih kursus dan memberi masukan bagi pengajar.

---

## Checklist Akun

- [ ] **Nama** sudah benar (untuk sertifikat)
- [ ] **Nomor WhatsApp** aktif (untuk undangan webinar)
- [ ] **Alamat pengiriman** terisi jika ada hadiah/merchandise
- [ ] Password kuat dan dijaga kerahasiaannya
- [ ] Mengetahui cara melanjutkan pembayaran dari **Pesanan Saya**
