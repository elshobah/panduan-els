# Menu Tags

Tags adalah kata kunci yang dikaitkan dengan kursus untuk membantu calon siswa menemukan kursus melalui hasil mesin pencari maupun filter di halaman kursus.

**Akses via:** WordPress Admin → **Tutor LMS** → **Tags**

---

## Tampilan Menu Tags

![Tampilan Menu Tags Tutor LMS](https://is3.cloudhost.id/elsweb/2022/07/image-3.png)

---

## Menambah Tag Baru

Di sisi kiri halaman, isi form berikut:

| Field | Keterangan |
|-------|-----------|
| **Nama** | Kata kunci tag (contoh: "javascript", "desain ui", "pemula") |
| **Slug** | URL-friendly version dari nama tag — dibuat otomatis |
| **Deskripsi** | Penjelasan singkat tentang tag ini |

Klik **Tambah Tag Baru** untuk menyimpan.

---

## Mengelola Tags yang Ada

Di daftar tag (sisi kanan), tersedia opsi:

- **Edit** — ubah nama, slug, dan deskripsi secara lengkap
- **Edit Cepat** — ubah nama dan slug langsung dari daftar
- **Hapus** — hapus tag (kursus yang menggunakan tag tidak ikut terhapus)
- **Lihat** — buka halaman arsip tag di frontend

---

## Tips Penggunaan Tags

{% hint style="success" %}
**Best practice tags:**
- Gunakan huruf kecil semua
- Spesifik tapi tidak terlalu panjang (1–3 kata)
- Konsisten — gunakan tag yang sama untuk topik yang sama
- Contoh: `html`, `css`, `pemula`, `desain-ui`, `bisnis-online`
{% endhint %}

Tags yang baik meningkatkan **discoverability** kursus Anda di halaman pencarian platform maupun Google.

---

## Perbedaan Tags dan Kategori

| | Kategori | Tags |
|-|---------|------|
| **Hierarki** | Bisa bertingkat (parent-child) | Flat, tidak bertingkat |
| **Jumlah per kursus** | Biasanya 1–2 | Bisa banyak |
| **Fungsi** | Pengelompokan utama | Kata kunci tambahan |
| **Contoh** | "Pemrograman Web" | "javascript", "react", "frontend" |
