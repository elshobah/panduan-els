# Tutorial Kompres Gambar

Gambar berukuran besar memperlambat loading website dan berdampak negatif pada SEO. Panduan ini menjelaskan dua cara mengompres gambar — offline menggunakan FastStone Image Resizer, dan online menggunakan berbagai tools gratis.

---

## Metode 1: FastStone Image Resizer (Offline)

FastStone Image Resizer adalah software desktop gratis untuk penggunaan non-komersial yang dapat memproses banyak gambar sekaligus (batch).

**Download:** [faststone.org/FSResizerDownload.htm](https://www.faststone.org/FSResizerDownload.htm)

### Langkah-Langkah

**1. Install & Buka Aplikasi**

Download, install, lalu jalankan FastStone Image Resizer.

**2. Masukkan Gambar**

Klik tombol **Add** untuk memilih gambar satu per satu, atau **Batch Convert** untuk memproses banyak gambar sekaligus.

**3. Atur Dimensi (opsional)**

| Opsi | Fungsi |
|------|--------|
| **Resize** | Ubah ukuran dalam piksel atau persentase |
| **Fit to** | Sesuaikan dengan dimensi tetap, menjaga proporsi |
| **Fill to** | Sesuaikan dimensi tanpa menjaga proporsi |

**4. Tentukan Kualitas**

Atur slider kompresi JPEG pada skala **1–100**:
- Nilai tinggi (80–95) = kualitas baik, ukuran lebih besar
- Nilai rendah (40–60) = ukuran kecil, kualitas menurun

**5. Pilih Format Output**

Pilih format: **JPEG** (foto), **PNG** (grafis/transparansi), BMP, dll.

**6. Tentukan Folder Penyimpanan**

Set lokasi folder output untuk hasil kompresi.

**7. Mulai Proses**

Klik tombol **Start** — FastStone akan memproses semua gambar sekaligus.

## Metode 2: Kompresi Online

Tidak perlu install — cukup buka browser dan upload gambar.

### Langkah-Langkah

1. Buka salah satu platform di bawah
2. Upload gambar Anda
3. Atur tingkat kompresi (jika tersedia)
4. Klik **Compress** / **Optimize**
5. Download hasilnya

### Platform Rekomendasi

| Platform | URL | Keunggulan |
|----------|-----|-----------|
| **TinyPNG** | [tinypng.com](https://tinypng.com/) | Terbaik untuk PNG & JPEG |
| **Compress JPEG** | [compressjpeg.com](https://compressjpeg.com/) | Khusus JPEG, simpel |
| **Optimizilla** | [imagecompressor.com](https://imagecompressor.com/) | Preview sebelum download |
| **ImageOptim** | [imageoptim.com](https://imageoptim.com/howto.html) | Lossless compression |
| **Kraken.io** | [kraken.io](https://kraken.io/plugins) | Integrasi plugin |

---

## Tips Memilih Metode

| Situasi | Rekomendasi |
|---------|-------------|
| Kompres banyak gambar sekaligus | FastStone Image Resizer |
| Kompres 1–5 gambar cepat | TinyPNG atau Optimizilla |
| Butuh kualitas maksimal | Lossless (PNG) via ImageOptim |
| Gambar foto/produk | JPEG dengan kualitas 70–85 |
| Gambar logo/ikon | PNG dengan TinyPNG |

{% hint style="success" %}
**Tips SEO:** Sebelum upload ke WordPress, kompres semua gambar terlebih dahulu. Gambar di bawah **150KB** adalah target yang baik untuk web.
{% endhint %}
