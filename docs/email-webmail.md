# Mengakses Email via Webmail

Webmail memungkinkan Anda mengakses email bisnis langsung dari browser, tanpa perlu menginstal aplikasi tambahan. Cocok digunakan dari komputer mana saja.

---

## Prasyarat

- Browser (Chrome, Firefox, Edge, Safari, dll.)
- **Alamat email bisnis** Anda
- **Password email** dari administrator

---

## Langkah-Langkah

### 1. Buka URL Webmail

Ketik alamat berikut di browser, ganti `domain.com` dengan domain Anda:

```
http://webmail.domain.com
```

atau

```
http://www.webmail.domain.com
```

**Contoh:** Jika domain Anda `tokoku.com`, aksesnya adalah `webmail.tokoku.com`

### 2. Login

Masukkan kredensial pada halaman login:

| Field | Isi dengan |
|-------|-----------|
| **Email** | Alamat email bisnis lengkap Anda |
| **Password** | Password email Anda |

Klik tombol **Login**.

### 3. Pilih Klien Email

Saat pertama kali login, sistem menampilkan pilihan klien email:

| Klien | Keterangan |
|-------|-----------|
| **Roundcube** | Modern, intuitif — **direkomendasikan** |
| **Horde** | Fitur lengkap, tampilan klasik |
| **Squirrelmail** | Ringan, minimal |

{% hint style="success" %}
Pilih **Roundcube** untuk pengalaman terbaik — antarmuka paling modern dan mudah digunakan.
{% endhint %}

### 4. Mulai Gunakan Email

Setelah memilih klien, Anda langsung masuk ke inbox dan dapat:

- Membaca email masuk
- Menulis dan mengirim email baru
- Mengatur folder dan filter
- Mengelola kontak

---

## Tips Penggunaan Roundcube

| Fitur | Cara Akses |
|-------|-----------|
| **Tulis email baru** | Klik tombol **Compose** |
| **Balas email** | Klik **Reply** pada email yang dibuka |
| **Buat folder** | Klik kanan di sidebar kiri → **Create Folder** |
| **Tandai penting** | Klik ikon bintang di sebelah email |
| **Cari email** | Gunakan kotak pencarian di bagian atas |

## Troubleshooting

**Halaman webmail tidak bisa dibuka?**
- Pastikan URL sudah benar (cek ejaan domain)
- Coba akses via HTTP jika HTTPS bermasalah
- Hubungi tim Elshobah jika masalah berlanjut

**Password salah?**
- Hubungi administrator website untuk reset password email
