## Langkah-Langkah

### 1. Buka Situs LMS

Buka [https://lms.elshobah.com](https://lms.elshobah.com) di browser. Anda akan melihat halaman utama. Klik **Masuk** (login) jika sudah punya akun, atau **Daftar** untuk membuat akun baru.

![Halaman utama LMS](https://GANTI-DENGAN-SCREENSHOT)

{% hint style="info" %}
Ganti setiap gambar `GANTI-DENGAN-SCREENSHOT` pada panduan ini dengan tangkapan layar asli, unggah ke penyimpanan gambar Anda, lalu tempel URL-nya.
{% endhint %}

### 2. Mendaftar Akun Baru

Buka halaman **Daftar Akun Baru** ([/register](https://lms.elshobah.com/register)) dan isi:

| Field | Keterangan |
|-------|-----------|
| **Nama Lengkap** | Nama Anda (akan tampil di sertifikat) |
| **Email** | Email aktif Anda |
| **Password** | Kata sandi |
| **Konfirmasi Password** | Ulangi kata sandi yang sama |

Klik **Daftar Sekarang**. 

Atau, lebih cepat, klik **Daftar dengan Google** untuk mendaftar memakai akun Google Anda tanpa mengisi form.

{% hint style="warning" %}
Halaman **/register** adalah untuk **peserta kursus**. Jangan tertukar dengan halaman **/daftar** yang merupakan pendaftaran **penyelenggara** (membuat lembaga LMS sendiri). Sebagai peserta, gunakan **/register**.
{% endhint %}

### 3. Masuk (Login)

Jika sudah punya akun, buka halaman **Masuk** ([/login](https://lms.elshobah.com/login)), masukkan **Email** dan **Password**, lalu klik **Masuk**. Anda juga bisa memakai **Masuk dengan Google**.

![Halaman login](https://GANTI-DENGAN-SCREENSHOT)

{% hint style="info" %}
Saat mendaftar atau login, mungkin muncul kotak verifikasi keamanan (anti-robot). Cukup tunggu sampai tercentang otomatis, atau ikuti petunjuk yang tampil. Ini normal untuk melindungi akun Anda.
{% endhint %}

## Mengenal Dashboard

Setelah masuk, Anda tiba di **Dashboard** dengan sapaan **"Selamat datang, {nama Anda}"**. Di sini terlihat ringkasan belajar Anda:

| Kartu | Arti |
|-------|------|
| **Total Kursus** | Jumlah kursus yang Anda miliki |
| **Sedang Berlangsung** | Kursus yang sedang dijalani |
| **Selesai** | Kursus yang sudah tuntas |
| **Total Donasi Saya** | Total donasi Anda (jika fitur donasi aktif) |

Di bawahnya ada bagian **Kursus Saya** dengan tombol **Mulai Belajar**, **Lanjut Belajar**, atau **Lihat Kembali** untuk tiap kursus.

![Dashboard peserta](https://GANTI-DENGAN-SCREENSHOT)

### Menu di Sidebar

Menu utama ada di sidebar kiri:

| Menu | Fungsi |
|------|--------|
| **Dashboard** | Halaman utama Anda |
| **Kursus Saya** | Kursus yang diikuti (Sedang Berlangsung / Selesai) |
| **Jelajahi Kursus** | Menemukan kursus baru |
| **Nilai & Sertifikat** | Gradebook dan sertifikat |
| **Ulasan Saya** | Ulasan yang Anda tulis |
| **Pesanan Saya** | Riwayat pembelian |
| **Langganan / Donasi / Afiliasi / Komunitas** | Fitur tambahan (bila diaktifkan lembaga) |
